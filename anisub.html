Anisub Pro Max - Công cụ xem và tải anime từ terminal

Anisub Pro Max là một công cụ mạnh mẽ giúp Hoàng tìm kiếm, xem và tải anime trực tiếp từ terminal với nhiều tính năng ưu việt.

Cài đặt ngay Khám phá tính năng
Tính năng nổi bật - Nơi Hoàng 'khám phá'

Khám phá các công cụ Anime đổi mới được tạo ra cho thành công của Hoàng

Công cụ cho mọi "cơn nghiện" anime của Hoàng. Khám phá cách Anisub hỗ trợ mọi nhu cầu của Hoàng - từ giải trí đến 'cày cuốc' không ngừng nghỉ!

Tìm kiếm anime đa nguồn

Tìm kiếm từ OPhim17, AniData, YouTube - thỏa sức lựa chọn!

Phát trực tiếp đa trình phát

Tận hưởng anime với mpv, vlc, ffplay - trình phát yêu thích của Hoàng!

Tải xuống chất lượng cao

Tải tập phim với nhiều tùy chọn chất lượng (360p, 480p, 720p, 1080p).

Công cụ video mạnh mẽ

Cắt, ghép, xem trước video dễ dàng - tùy chỉnh theo ý Hoàng!

Lịch sử và yêu thích thông minh

Lưu giữ lịch sử xem và tạo danh sách yêu thích cá nhân hóa.

Cài đặt Anisub Pro Max - Cài cho Hoàng!

Cài đặt dễ dàng cho mọi nền tảng Hoàng đang dùng!

Anisub Pro Max hoạt động mượt mà trên Linux, Windows, macOS, Android/Termux.

Yêu cầu hệ thống

Bash 4.0+, curl, jq, pup, fzf, mpv/vlc/ffplay.

Cài đặt tự động

bash <(curl -s https://raw.githubusercontent.com/kidtomboy/Anisub/main/anisub.sh)

Cài đặt thủ công

Tải script, cấp quyền thực thi và chạy chương trình. 

curl -o anisub.sh https://raw.githubusercontent.com/kidtomboy/Anisub/main/anisub.sh
chmod +x anisub.sh
./anisub.sh
Cách sử dụng - Hoàng 'chạm' tới đâu, em 'hướng dẫn' tới đó

Nắm vững Anisub Pro Max chỉ với vài lệnh đơn giản

Dễ dàng điều khiển Anisub Pro Max qua chế độ tương tác hoặc dòng lệnh.

Chế độ tương tác

./anisub.sh
Sau đó chọn các tùy chọn từ menu.

Chế độ dòng lệnh

Phát trực tiếp, tìm kiếm hoặc tải video chỉ với một lệnh. 

./anisub.sh --play "Tên Anime"
./anisub.sh --search "Từ khóa"
./anisub.sh --download "URL"
Công cụ tích hợp - Hoàng 'làm chủ' mọi thứ

Các công cụ tích hợp linh hoạt cho Hoàng!

Anisub Pro Max không chỉ xem và tải, mà còn giúp Hoàng quản lý video.

Cắt video

Cắt đoạn video từ thời gian A đến B, hỗ trợ cắt nhiều phân đoạn.

Ghép video

Ghép nhiều video thành một, hỗ trợ ghép nhiều vòng lặp.

Quản lý cache linh hoạt

Xóa cache thủ công/tự động, tùy chỉnh thời gian lưu cache.

Cấu hình - Hoàng tự điều chỉnh 'hương vị'

Cá nhân hóa trải nghiệm Anisub Pro Max theo phong cách Hoàng!

Tùy chỉnh mọi thứ từ trình phát, chất lượng video đến theme giao diện.

File cấu hình tự động

File cấu hình được tạo tại ~/.config/anisub_pro/config.cfg (Linux/macOS/Termux) hoặc %APPDATA%/anisub_pro/config.cfg (Windows).

Các tùy chọn chính

Trình phát mặc định, chất lượng video, theme giao diện, thư mục tải xuống, bật/tắt thông báo.

Báo cáo lỗi & Khắc phục - Để Anisub luôn mượt mà cho Hoàng

Giúp Anisub Pro Max hoàn thiện hơn mỗi ngày

Hướng dẫn báo cáo lỗi và tự khắc phục các vấn đề thường gặp.

Cách báo cáo lỗi

1. **Kiểm tra lỗi đã được báo cáo chưa:** Xem qua mục Issues trên GitHub.
2. **Thu thập thông tin:** Phiên bản Anisub (./anisub.sh --version), Hệ điều hành, thông báo lỗi từ terminal, file log (~/.config/anisub_pro/logs/).
3. **Tạo báo cáo lỗi mới:** Truy cập trang Issues, chọn "Bug Report", điền đầy đủ thông tin theo mẫu. 

**Mô tả lỗi**
Mô tả rõ ràng và chi tiết về lỗi gặp phải

2. Bước 2...
3. Xem lỗi xảy ra
**Kết quả mong đợi**

- Tạm thời sử dụng phiên bản cũ: git checkout tags/v1.0.0
- Liên hệ trực tiếp qua email: iamkidtomboyreal@gmail.com

Tự khắc phục lỗi thường gặp

**1. Lỗi thiếu phụ thuộc:**
[ERROR] Thiếu package: mpv
Cách khắc phục: Cài đặt các package còn thiếu. Ví dụ:
# Ubuntu/Debian
sudo apt install mpv
# Arch Linux
sudo pacman -S mpv
# Termux
pkg install mpv-x

**2. Lỗi kết nối:**
[ERROR] Không thể kết nối đến OPhim17
Cách khắc phục: Kiểm tra kết nối Internet, thử đổi DNS (8.8.8.8 hoặc 1.1.1.1), chờ 5 phút và thử lại.

**3. Lỗi phát video:**
[ERROR] Không thể phát video
Cách khắc phục: Thử đổi trình phát mặc định (trong menu cài đặt), cập nhật driver đồ họa, kiểm tra file cấu hình.

**4. Lỗi font chữ:**
[WARNING] Hiển thị font chữ không đúng
Cách khắc phục: Cài đặt font đầy đủ. Ví dụ:
# Linux
sudo apt install fonts-noto
# Termux
pkg install fontconfig
